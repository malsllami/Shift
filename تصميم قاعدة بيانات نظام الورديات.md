# تصميم قاعدة بيانات نظام الورديات

## القواعد الأساسية

-   قاعدة بيانات رئيسية واحدة فقط.
-   ملفات الورديات والإداري عبارة عن جسور عرض وتصدير.
-   جميع أسماء الجداول والأعمدة باللغة العربية.
-   بدون شرطات أو رموز.
-   المسافة بين الكلمات مسافة واحدة.
-   لا توجد أي بيانات ثابتة داخل الأكواد.
-   جميع القيم تقرأ من جدول الإعدادات.
-   جميع التواريخ تكون ميلادي وهجري.
-   الأرقام العربية تحول إلى إنجليزية عند الإدخال والحفظ.
-   تحديث ملفات الجسر يكون فوريا بعد التعديل.
